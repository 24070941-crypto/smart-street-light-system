# Smart Street Light System (Arduino)

This project is an IoT-based smart street lighting system that automatically controls street lights based on environmental conditions.

## Features
- Automatic ON/OFF using LDR sensor
- Motion detection using IR sensor
- Energy saving up to 40%
- Low-cost and efficient design

## Components Used
- Arduino Uno
- LDR (Light Dependent Resistor)
- IR Sensor
- LEDs
- Breadboard & Wires

## Working
The system detects ambient light using LDR. During night, lights turn ON at low intensity. When motion is detected, lights increase brightness and then return to normal after inactivity.

## Applications
- Smart cities
- Energy-efficient street lighting
- IoT-based automation
